Box Shadow digunakan untuk mengatur bayangan pada sebuah box

dibwah ini yang wjib

  • x-offset -> mengatur jarak bayangan smbu x
  • y-offset -> mengatur jarak bayangan smbu y
  • blur -> mengatur tingkat blur
  • color -> mengatur twarna (biasa menggunakan opacity dengan RGBa)

box-shadow: (x-offset) (y-offset) (blur) (color)

box-shadow: 20px 0 20px rgb(9, 9, 9, 80%)

Optional

(spread)

spread untuk menambah ukuran dari shadow

box-shadow: (x-offset) (y-offset) (blur) (spread) (color)

box-shadow: 20px 0 0 rgb(9, 9, 9, 80%);

box-shadow: 20px 0 0 10px rgb(9, 9, 9, 80%);

nilai spread bisa negatif (membuat bayangan lebih kecil dari shadow)

box-shadow: 20px 0 10px -10px rgb(9, 9, 9, 80%);

(inset)

inset membuat bayangan ada di dalam elemen

box-shadow: (inset) (x-offset) (y-offset) (blur) (color)

box-shadow: inset -20px 0 20px rgb(9, 9, 9, 80%);

multiple shadow

kita bisa memberikan lebih dari satu bayangan dalam sebuah element

box-shadow: (x-offset) (y-offset) (blur) (color), (x-offset) (y-offset) (blur) (color), dst

kita juga bisa membuat layaknya border yang bertumpuk tumpuk

semua bisa digabung dengan hover
