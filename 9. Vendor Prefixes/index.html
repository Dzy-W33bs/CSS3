vendor prefixes

teknik yang digunakan agar syntax css 3 yang kita buat kompatibel di semua browser

vinder prefixes -> sintaks khusus yang harus ditambahkan untuk beberapa property CSS3. 

        -webkit-(properti-css3)  -> chrome, safari, opera versi terbaru
        -moz-(properti-css3)     -> firefox
        -ms-(properti-css3)      -> internet explorer
        -o-(properti-css3)       -> opera versi terdahulu
