radial gradient

radial gradient memberikan gradasi warna secara melingkar


background-image: radial-gradient(warna1, warna2,......, warna-n)

syntax lengkap

radial-gradient(
tipe jangkauan at posisi-y posisi-x,
warna1,
warna2,
warna-n
);

tipe

  • ellipse (default)
  • circle

jangkauan -> jarak dari pusat gradientke sisi terluarnya

  • closest-corner -> jarak dari pusat gradient ke sudut terdekat elemen
  • closest-side -> jarak dari pusat gradient ke sisi terdekat elemen
  • farthest-corner -> jarak dari pusat gradient ke sudut terjauh elemen
  • farthest-side -> jarak dari pusat gradient ke sisi terjauh elemen

posisi-y

  • top
  • center
  • bottom

posisi-x

  • left
  • center
  • right
