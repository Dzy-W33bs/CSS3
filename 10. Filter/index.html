filter

filter adalah properti CSS3 yang digunakan untuk memberikan efek visual pada gambar, background, atau border

efek filter yang bisa digunakan: 

        blur (px)
        brightness (angka / %)
        contrast (angka / %)
        saturate (angka / %)
        grayscale (angka / %)        
        sepia (angka / %)        
        hue-rotate (adeg)        
        invert (angka / %)        
        opacity (angka / %)   -> sama saja dengan properti opacity   
        drop-shadow (sama seperti box-shadow)

original

blur

brightness

nilai default 100%

contrast

default 100%

saturate

default 100%

grayscale

default 0

sepia

default 0

hue-rotate

pake deg(derajat)

invert

default 0

opacity

sama saja dengan properti opacity

drop-shadow

sama saja dengan properti box-shadow. bedanya kalau filter bisa memberikan bayangan pada gambar png pada element yang tidak transparan
