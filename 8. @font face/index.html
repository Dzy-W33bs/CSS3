@font-face

@font-face digunakan untuk mengubah font

caranya:

  1. kalian harus punya dulu fontnya
  2. simpan di folder website
  3. di css paling atas tulis syntaxnya

syntax

        @font-face {
            font-family: 'nama font baru';
            src: url('folder/fontbaru.eot');
            src: url('folder/fontbaru.eot?#iefix') format('embedded-opentype'),
            url('folder/fontbaru.woff2') format('woff2'),
            url('folder/fontbaru.woff') format('woff'),
            url('folder/fontbaru.ttf') format('truetype'),
            url('folder/fontbaru.svg#svgFontName') format('svg'),
        }

yang paling bagus untuk web adalah font type woff dan woff2

woff2 dan woff digunakan bersamaan dengan woff2 terlebih dahulu

font tipe ttf dan otf bisa diubah ke woff dan woff2

tempat rekomended download font dan ubah ttf/otf ke woff dan woff2: fontsquirrel.com
