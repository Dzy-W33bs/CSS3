transform

properti transform memungkinkan kita dapat memanipulasi format visual dari elemen HTML

fungsi transform

  • scale memperbesar/memperkecil
  • rotate memutar
  • skew membuat condong/miring
  • translate mengubah posisi
                scale (angka), scaleX (angka), scaleY(angka) 
		rotate (deg), rotateX (deg), rotateY(deg) 
		skew(deg), skewX (deg), skewY (deg) 
		translate(px), translateX(px), translate(px)

akan kuberikan contoh berupa hover+transition agar mudah dilihat

scale

transform: scale(2,.5); dibaca: perbesar sumbu x 2 kali lipat, perbesar sumbu y sebesar 0,5 kali lipat

rotate

transform: rotate(180deg);

transform: rotateX(180deg);

transform: rotateY(180deg);

semuanya bisa digabung

transform: rotate(180deg) rotateX(180deg) rotateY(180deg);

skew

skew = skew x

transform: skew(180deg);

transform: skewY(60deg)

translate

transform: translate(100px);

transform: translate(400px, -200px);

apa bedanya dengan position absolut?

efek yang dihasilkan sama saja tetapi menggunakan translate performanya jauh lebih baik daripada menggunakan position
