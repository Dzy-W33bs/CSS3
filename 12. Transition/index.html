transition

properti transition memungkinkan kita dapat mengubah nilai dari property HTML secata halus

perbadaan dari property transition dan animation yaitu kalau transition hanya ada dua keadaan, yaitu keadaan awal dan akhir

transition disimpan di keadaan awal/bukan pada keadaan akhir (misal hover)


transition: property durasi(ini wajib ada) fungsi delay

durasi

s/ms

kalau ingin mengubah element HTML dari kondisi awal ke akhir ingin berapa lama

1s
5s

property

menentukan properti apa saja yang akan dikenai transisi

fungsi

fungsi transisi/timing function

  • ease
  • ease-in
  • ease-out
  • ease-in-out
  • linear
  • cubic-bezier (w,x,y,z) -> bisa menggunakan ispect sebagai bantuan (lihat unpas pes transition part 2)

delay

angka setelah durasi. misal 3s 2s (maka delaynya adalah 2s)

artinya tunggu dulu 2s (delay) sebelum melakukan transisi selama 3s
